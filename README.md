# Requirements

* Clone this repository
* Install NodeJs :

    `sudo apt install nodejs`
* Install expo :

    `npm install expo-cli --global `

# How to launch the project

* cd to repo
* Install dependencies:

    `npm install`
* Run build 

    ```npm start```
* Start expo force tunnel
   
    ```expo start --tunnel```
* Get the QR Code from http://localhost:19002/
